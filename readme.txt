== Description ==

CRM LastPosts Widget shows the Last Posts, the Most Popular Posts and Random Posts in a visual way using the thumbnail of every post. It shows the post title and date when you mousover the image. In the Popular Posts mode you see the post number aswell.

You can choose order by date, most popular posts or random order.

Send your questions or bugs reports to: apps@cromorama.com

CRM LastPosts Widget es un sencillo widget para WordPress que muestra los últimos posts, los posts mas populares y posts aleatorios de una manera muy visual usando la imagen destacada de cada post como “ventana” al mismo y muestra el título y la fecha al situarnos encima con el ratón creando un efecto de difuminado en la imagen.

El proyecto comenzó como una necesidad de Cromorama que hemos decidido liberar para que todo el mundo pueda disfrutarlo de manera gratuita. 

Mas información en: http://www.cromorama.com/blog/crm-lastposts-widget/. Iremos actualizando y mejorando la funcionalidad del mismo con nuevas versiones, si deseas comentarnos algo acerca del proyecto puedes hacerlo aquí: apps@cromorama.com 

== Installation ==

1. Upload the crm-lastposts folder to Plugins folder of your WordPress.
2. Activate widget from plugins page.
3. Configurate your widget and enjoy it!.
