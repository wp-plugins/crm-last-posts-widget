== Description ==

CRM LastPosts Widget is a widget that shows the last posts in a visual way using the thumbnail of every post. It shows the post title and date when you mousover the image. Results are filter by date so you see the last posts first.

Support: apps@cromorama.com

CRM LastPosts Widget es un sencillo widget para WordPress que muestra los últimos posts de una manera muy visual usando la imagen destacada de cada post como “ventana” al mismo y muestra el título y la fecha al situarnos encima con el ratón creando un efecto de difuminado en la imagen.

Los resultados son mostrados de forma descendiente ordenados por fecha, de forma que salgan los mas recientemente publicadas primero.

El proyecto comenzó como una necesidad de Cromorama que hemos decidido liberar para que todo el mundo pueda disfrutarlo de manera gratuita. 

El widget está disponible desde el repositorio de WordPress así como para descargar desde http://www.cromorama.com/blog/crm-lastposts-widget/. Iremos actualizando y mejorando la funcionalidad del mismo con nuevas versiones, si deseas comentarnos algo acerca del proyecto puedes hacerlo aquí: apps@cromorama.com 

== Installation ==

1. Upload the crm-lastposts folder to Plugins folder of your WordPress.
2. Activate widget from plugins page.
3. Configurate your widget and enjoy it!.
